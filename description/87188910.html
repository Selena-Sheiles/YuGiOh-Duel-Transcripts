Card type Synchro Monster

Level 9

[Fish / Synchro / Effect]
1 Tuner + 1+ non-Tuner monsters

You can only use effect ① and ③ of "Ravenous Crocodragon Archethys" once per turn.

①: If this card is Synchro Summoned: You can draw cards equal to the number of non-Tuners used as Synchro Materials.

②: Gains 500 ATK/DEF for each card in your hand.
